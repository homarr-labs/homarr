The Anchor Note widget shows a selected note from your Anchor integration. If your note permissions allow it, you can also edit the note title and content directly in Homarr.

### Supported Integrations

<WidgetIntegrations items={[{
  integration: anchorIntegration
}]} />

### Interactions

- Click **Edit** to start editing the selected note.
- Click **Save** to persist changes to Anchor.
- Click **Cancel** to discard unsaved changes.
