The VPN widget provides a compact overview of each configured VPN integration. It shows whether the tunnel and DNS
services are up, the currently exposed public IP address, and the VPN provider and protocol being used.

When multiple VPN integrations are attached, the widget renders one card per integration so you can compare their
current status at a glance.
