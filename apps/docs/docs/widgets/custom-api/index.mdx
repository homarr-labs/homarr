This widget fetches data from a custom API endpoint defined in [Custom Widgets management](/docs/management/custom-widgets) and renders it using configurable display types.

Custom API widgets are created and configured through the admin management page. Each widget definition specifies the API endpoint, authentication, display type, and refresh interval.

See the [Custom Widgets documentation](/docs/management/custom-widgets) for full details on creating and configuring custom widgets.

### Adding the widget

<AddingWidget />

:::note
Custom API widgets must first be created in **Management → Custom Widgets** before they appear in the widget picker.
:::
