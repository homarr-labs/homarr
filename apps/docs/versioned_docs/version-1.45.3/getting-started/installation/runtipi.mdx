#  Runtipi

Runtipi is a self-hosted orchestrator that makes it easy to manage and run multiple services on a single server. It is based on Docker and comes with a simple web interface to manage your services. Runtipi is designed to be easy to use, so you don't have to worry about manual configuration or networking.

https://runtipi.io

## Prerequisites

- [Runtipi](https://runtipi.io/docs/getting-started/installation)

# Installation

Installing Homarr is pretty untuitive, the app is available in the default appstore :
1. Search for **"Homarr"** in the App-Store page and select it
2. Click on **"Install"**.
3. Choose the method you want to access Homarr, through any port or domain name then select **"Install"**

# Maintenance

After installation, all you need is available from the WebUI in **"My apps"** > **"Homarr"** :
 - Start/Stop
 - Update
 - Backup
 - Logs
 
If you like to tinker with your apps, you can also edit your app's docker-compose with a [user-config](https://runtipi.io/docs/guides/customize-app-config).
