## ![](img/headers/features.png)

Homarr is a powerful, customizable dashboard that brings all your self-hosted services together in one beautiful interface. Built with modern technologies and designed for ease of use.

- **üñåÔ∏è Highly Customizable** - Extensive drag-and-drop grid system for complete layout control
- **üîó Seamless Integrations** - Connect with your favorite self-hosted applications effortlessly
- **‚ö° Fast App Management** - Intuitive interface with no YAML configuration required
- **üë• Advanced User Management** - Comprehensive user management with permissions and groups
- **üîê Enterprise Authentication** - Single sign-on support via OIDC and LDAP
- **üîí Secure by Default** - Industry-standard encryption using BCrypt and AES-256-CBC
- **‚ö° Real-time Updates** - Live widget updates powered by WebSockets, tRPC, and Redis
- **üîç Powerful Search** - Fast built-in search across thousands of data points and integrations
- **üé® Extensive Icon Library** - Icon picker with over 11,000 icons to choose from
- **üñ•Ô∏è Universal Compatibility** - Runs on any major hardware (x86, ARM, Raspberry Pi) and OS (Windows, Linux, TrueNAS, Unraid)
- **‚ò∏Ô∏è Kubernetes Ready** - Full Helm chart support for efficient scaling and high reliability
